// Тип данных char
quest1.c - Сопоставьте символ с его номером в таблице ASCII.
quest2.c - Написать программу для перевод символов в верхний регистр.
quest3.c - Вывести соседние символы в таблице ASCII.
quest4.c - Написать программу для подсчёта различных символов, которые поступают на вход.
quest5.c - Определить тип символа.
quest6.c - Вывести на экран KK символ латинского алфавита в верхнем и нижнем регистре.
quest7.c - Вывести на экран последние K символов латинского алфавита в верхнем регистре в алфавитном порядке.

// Символьные строки
quest8.c - Подсчитать количество символов в строке, без учёта нуль-символа.
quest9.c - Подсчитать количество слов в строке. Под словом будем понимать непустую последовательность символов, разделённую знаками пробела.
quest10.c - Поиск символа. Найти в строке заданный символ.
quest11.c - Системы счисления. Дана строка, изображающая двоичную запись целого положительного числа. Вывести строку, изображающую десятичную запись этого же числа.
quest12.c - Лишние пробелы. Удалить из строки лишние пробелы.
quest13.c - Расстояние Хэмминга.
quest14.c - Напишите программу, которая по описанию пути к кладу определяет точные координаты клада, считая, что начало координат находится в начале пути, ось OX направлена на восток, ось OY – на север.
quest15.c - Удалить часть символьной строки, заключенную в комментарий
quest16.c - Написать программу, реализующую вывод числа с разделением на триады цифр
quest17.c - Дана строка. Определить является ли данная строка записью числа (целого или вещественного). Дробная и вещественная части разделены десятичной точкой.
quest18.c - Дана строка. Найти в ней слова, которые начинаются и оканчиваются одной и той же буквой. Слова разделяются пробелами.
quest19.c - Шифровка. В строке слова зашифрованы — каждое из них записано наоборот. Написать программу, расшифровывающую строку.

// Стандартные функции работы со строками.
quest20.c - Программа должна склеивать строку, которая уже есть в программе и строку, введенную пользователем.
quest21.c - Вывести полное название файла в формате
quest22.c - Определить можно ли из указанных слов составить цепочку для игры в "Города". Т.е. либо первая буква второго слова совпадает с последней буквой первого слова, либо наоборот. Названия городов, как и полагается, записаны с заглавной буквы.
quest23.c - Даны две строки. Необходимо сравнить строки между собой с учётом регистра символов.
quest24.c - Даны две строки. Необходимо сравнить строки между собой без учёта регистра символов.
quest25.c - Напишите программу проверки того, что число, записанное в строке, является номером счастливого билетика. Билет называется счастливым, если сумма первых трёх цифр номера билета равняется сумме последних трёх цифр номера билета.
quest26.c - Написать программу, проверяющую, можно ли из букв, входящих в первую строку, составить вторую строку. (буквы можно использовать не более одного раза и можно переставлять)
quest27.c - Написать программу, которая переводила бы строку на английском языке в морзянку.
