// Функции и переменные
quest1.c - Напишите функцию с именем sum, которая вычисляет сумму двух, переданных в неё целочисленных аргументов.
quest2.c - Опишите функцию с именем min, которая вычисляет минимальное из двух чисел, переданных в неё.
quest3.c - Напишите программу, которая будет вычислять и возвращать минимальное из трёх чисел, переданных в неё, при помощи функции min.
quest4.c - Напишите функцию с именем factorial, которая вычисляет факториал числа, переданного ей в качестве аргумента.
quest5.c - Допустим, у вас имеется N хоккеистов. Из них нужно сформировать команду из K человек. Сколько существует вариантов это сделать?
quest6.c - Напишите функцию с именем sequence_multipliers, принимающую один аргумент x, которая определяет можно ли представить x в виде произведения трех последовательных натуральных чисел. Например, 120=4⋅5⋅6
quest7.c - Напишите функцию с именем square(n,c), которая выводит на экран квадрат размера n на n, заполненный символами c.
quest8.c - Форматирование даты
quest9.c - Написать функцию выводящую все делители числа K, поступившего на вход.
quest10.c - Переделайте функцию, написанную в прошлом уроке, таким образом, чтобы она возвращала число 1, если число K простое, или 0, если число K составное.

// Указатели
quest11.c - В программе имеются две целочисленные переменные a и b. Объявите два указателя с именами p_a и p_b, которые ссылаются соответствующие переменные.
quest12.c - В программе имеются символьные переменные ch и inp. Объявите  указатель с именем p_ch, который ссылается на эту переменную
quest13.c - В программе объявлены две переменные целого типа. На каждую из них ссылается указатель. На первую ссылается указатель p_1, а на вторую указатель p_2. Используя указатели, вывести значения этих переменных через пробел. Сначала вторую, потом первую.
quest14.c - Выведите сумму переменных, на которые ссылаются данные указатели.
quest15.c - В программе объявлены две переменные целого типа. На каждую из них ссылается указатель. На первую ссылается указатель p_1, а на вторую указатель p_2. Кроме того, объявлена переменная типа double и указатель на неё p_dbl. Используя указатели, подсчитать частное целых переменных (первую делим на вторую) и сохранить это значение в переменную, на которую ссылается p_dbl.
quest16.c - В программе объявлены два указателя p_1 и p_2, которые ссылаются на некоторые переменные. Присвойте переменной на которую ссылается указатель p_1, значение переменной c, а переменной, на которую ссылается указатель p_2, значение переменной d.
