// Одномерные массивы
quest1.c - Переставить элементы массива в обратном порядке
quest2.c - Определить является ли массив палиндромом, т.е. первый элемент равен последнему, второй предпоследнему и т.д.
quest3.c - Вывести сначала чётные элементы массива, а затем нечётные.
quest4.c - Напишите программу, выводящую на экран все элементы массива, которые меньше последнего.
quest5.c - Напишите программу, выводящую на экран все элементы массива, которые меньше последнего элемента и больше первого элемента массива.
quest6.c - Найти среднее арифметическое элементов массива.
quest7.c - Разделить массив на две части, поместив в первую элементы, большие среднего арифметического массива, а во вторую — меньшие.
quest8.c - В заданном массиве поменять местами первую и вторую половины
quest9.c - В заданном массиве поменять местами наибольший и наименьший элементы.
// Двумерные массивы
quest10.c - Вывести элементы двумерного массива (матрицы) на экран.
quest11.c - Для заданной матрицы вывести на экран транспонированную матрицу.
quest12.c - По заданному числу N сформировать матрицу (1234-4321)
quest13.c - По заданному числу N сформировать матрицу (1234-2123-3213-4321)
quest14.c - Для заданной квадратной матрицы посчитать произведение на главной диагонали и отдельно побочной
quest15.c - Для заданной квадратной матрицы A[N] посчитать величины: сумму элементов, находящихся над главной диагональю и сумму элементов, расположенных под побочной диагональю
quest16.c - Для заданной матрицы посчитать сумму элементов каждого столбца.
quest17.c - Поменять местами столбцы с минимальной и максимальной суммой элементов.
quest18.c - Произвести циклический сдвиг столбцов квадратной матрицы A[N] на KK позиций вправо.
quest19.c - Змейка. Для заданных чисел M и N сформировать матрицу, заполненную по спирали.
