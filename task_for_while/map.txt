quest1.c - Написать программу выводящую на экран первые N натуральных чисел.
quest2.c - Написать программу выводящую на экран все натуральные числа из промежутка [K, M].
quest3.c - Написать программу выводящую на экран все натуральные числа из промежутка [K, M]. Дополнительно на отдельной строке вывести количество напечатанных чисел.
quest4.c - Вывести через пробел все целые числа из промежутка [A,B] в обратном порядке.
quest5.c - Посчитать факториал числа.
quest6.c - Вывод числа Фибоначчи.

quest8.c - Вывести числа из промежутка [A;B]. При этом число A должно только A раз, число A+1 - A+1 раза, A+2 - A+2 раза и т.д.
quest9.c - Для заданного числа number вывести все его делители и их количество на экран
quest10.c - Для заданного числа NN проверить, является ли оно простым
quest11.c - Программа, которая выводит максимальное число среди введенных (исправить код)
